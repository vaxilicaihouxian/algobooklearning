# 重写 ENQUEUE 和 DEQUEUE,使之能够处理上溢和下溢

原
```
ENQUEUE(Q,x):
  Q[Q.tail] = x
  if Q.tail == Q.length:
      Q.tail = 1
  else:
    Q.tail = Q.tail + 1
DEQUEUE(Q):
  x = Q[Q.head]
  if Q.head == Q.length
    Q.head = 1
  else:
    Q.head = Q.head+1
  return x
```

新
```
ENQUEUE(Q,x):
  if Q.tail == Q.head:
    return "stack overflow"
  Q[Q.tail] = x
  if Q.tail == Q.length:
      Q.tail = 1
  else:
    Q.tail = Q.tail + 1
DEQUEUE(Q):
  if Q.head == Q.tail:
    return "stack overflow"
  x = Q[Q.head]
  if Q.head == Q.length
    Q.head = 1
  else:
    Q.head = Q.head+1
  return x
```
